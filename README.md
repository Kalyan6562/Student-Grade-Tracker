# Student Grade Tracker

A Java Swing-based application for managing student records, calculating averages, and assigning grades.

## Features

- Add student details
- Enter marks for 5 subjects
- Automatic average calculation
- Grade assignment (A, B, C, D, F)
- View all student records
- Sort and display marks using LinkedList
- GUI built using Java Swing

## Technologies Used

- Java
- Java Swing
- ArrayList
- LinkedList
- Collections Framework
- OOP Concepts

## Grading System

| Average | Grade |
|----------|--------|
     90+        A 
    75-89       B 
    60-74       C 
    40-59       D 
   Below 40     F 

## How to Run

1. Compile:
   ```bash
   javac finalproject.java
   ```

2. Run:
   ```bash
   java finalproject
   ```
